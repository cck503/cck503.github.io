寫下自己的個人介紹

讓更多的人瞭解你

世界上有一種職業工時很長、沒有薪水、狀況很多、責任很大、沒有任何商業模式、算不出什麼獲利方程式，每天工作24小時且年中無休，這個職務叫做…「媽媽」。

小孩的成長直接與媽媽人生畫上了等號！雖說一切都是甜蜜的負擔，背後有許多許多犧牲與奉獻是確實存在的。

BabyHome的價值主張是珍惜媽媽的分享、創造溫暖的價值！提供媽媽在育兒、持家的基本資訊，溫暖的平台陪伴著「家」的喜怒哀樂。最重要是支持媽媽實現自己的「夢想」、成就他人的「希望」。

傾聽、學習、傳承是我們一直堅持的理念！希望透過無國界的網路，將「台灣好媽媽」滿滿的愛與能量溫暖地球每一個角落。

BabyHome的婦幼資訊廣度及深度兼具，爸媽們在此交流豐富深入，網友閱讀停留時間長，是一個高人氣且樂於分享養兒育子經驗的活力網站。BabyHome之所以會日漸成長茁壯，都是因為背後這群龐大的會員無私、樂意的提供意見，支援著我們往前走，還可以藉此針對會員的意見不斷開發新的服務，使網站形態從最初的部落格進化，到今日從供給到雙向互動的完整平臺，有家族、討論區、專業醫生駐站解決會員的大小事、親子知識性文章的分享等一應俱全。
